Käsikonsoleita Seuraava konsoli

Game Boy Color (jap. ゲームボーイカラー) on Nintendon valmistama ja julkaisema käsipelikonsoli, jonka edeltäjä oli suosittu Game Boy. Se julkaistiin Japanissa 21. lokakuuta 1998, Pohjois-Amerikassa 19. marraskuuta 1998, Euroopassa 23. marraskuuta 1998 ja Isossa-Britanniassa 27. marraskuuta 1998. Game Boy Color on ohuempi kuin Game Boy Pocket ja siinä on värinäyttö. Kuten alkuperäisessä Game Boyssa, siinä on 8-bittinen suoritin. Laitteessa toimivat myös alkuperäisen Game Boy -mallin pelit, joihin on jopa joissakin tapauksissa suunniteltu värit etukäteen. Game Boyta ja Game Boy Coloria on myyty yhteensä 118,7 miljoonaa yksikköä.

Game Boy Color oli vastaus pelinkehittäjien painostukseen luoda uusi ja paljon kehittyneempi käsikonsoli, koska heidän mielestään Game Boy ja sen uudistettu versio Game Boy Pocket olivat ominaisuuksiltaan riittämättömiä. Kehityksen tuloksena syntyi maailman ensimmäinen taaksepäin yhteensopiva käsikonsoli; koska Game Boy Colorilla pystyi pelaamaan alkuperäisen Game Boyn pelejä, sen pelivalikoima oli suurempi ja siten myös potentiaalisten käyttäjien määrä lukuisampi. Taaksepäin yhteensopivuudesta tuli Game Boy Colorin jälkeen Game Boy -konsolisarjan tärkeä ominaisuus, koska se mahdollisti julkaisuvaiheessa suuremman pelikirjaston kuin kilpailevissa konsoleissa.

Prosessorin, joka on Sharpin valmistama muunneltu Zilog Z80, kellotaajuus on noin 8 MHz eli kaksinkertainen alkuperäiseen Game Boyhin verrattuna. Game Boy Colorissa oli myös neljä kertaa alkuperäisen Game Boyn verran muistia (32 kilotavua keskusmuistia, 16 kilotavua videomuistia). Näytön resoluutio on sama kuin Game Boyssa eli 160×144 pikseliä.

Game Boy Colorissa oli myös infrapunayhteys langatonta kommunikointia varten. Tätä ominaisuutta käyttivät kuitenkin vain muutamat pelit, ja se jätettiin pois Game Boy Advancesta ja myöhemmistä Game Boy -konsoleista. Game Boy Colorin väripaletissa oli 32 768 väriä, joista 56 kyettiin käyttämään samanaikaisesti näytöllä, minkä lisäksi konsoli pystyi käyttämään neljän värin perusvarjostusta Game Boylle kehitettyihin peleihin. Game Boy Color pystyi värjäämään sprite-grafiikan ja taustan erikseen, mutta tämä aiheutti graafisia häiriöitä joissain peleissä.

Erityisesti Game Boy Colorille suunniteltujen pelien kotelot ovat väriltään kirkkaita. Ne eivät ole yhteensopivia uudemman Nintendo DS:n kanssa. Game Boyn kanssa yhteensopivat Game Boy Colorille suunnitellut pelit ovat muodoltaan samankaltaisia kuin Game Boyn harmaat pelimoduulit, mutta väriltään mustia tunnistamisen helpottamiseksi. Pokémon Yellowin ja Pokémon Goldin ja Silverin pelimoduulit poikkeavat väreiltään tavanomaisista moduuleista, mutta ovat teknisesti identtisiä niiden kanssa.
